The folder has scattered AI work, old sessions, project notes, memory files, recovered transcripts, prototypes, exports, and even a few unrelated projects.

Her answer was not “let me reorganize it for you.”

It was: build a read-only map first.

No files moved. No files renamed. No cleanup pass. Just scan what's there, preserve the original paths, detect project clusters, show the evidence, and suggest possible structures before I choose what happens next. That's today's build-in-public artifact for Mira-OS:
